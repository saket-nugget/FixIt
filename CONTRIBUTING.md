# Contributing to FixIt

Thank you for your interest in contributing to FixIt! We welcome contributions from everyone.

## Code of Conduct

By participating in this project, you agree to abide by our Code of Conduct. Please treat everyone with respect and kindness.

## How to Contribute

1.  **Fork the repository**: Click the "Fork" button at the top right of this page.
2.  **Clone your fork**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/FixIt.git
    cd FixIt
    ```
3.  **Create a branch**:
    ```bash
    git checkout -b feature/amazing-feature
    ```
4.  **Make your changes**: Implement your feature or fix.
5.  **Commit your changes**:
    ```bash
    git commit -m "Add some amazing feature"
    ```
6.  **Push to your fork**:
    ```bash
    git push origin feature/amazing-feature
    ```
7.  **Open a Pull Request**: Go to the original repository and click "New Pull Request".

## Reporting Bugs

If you find a bug, please open an issue using the Bug Report template. Include as much detail as possible:

*   Steps to reproduce
*   Expected behavior
*   Actual behavior
*   Screenshots (if applicable)

## Feature Requests

Have an idea? We'd love to hear it! Open an issue using the Feature Request template to describe your idea.
